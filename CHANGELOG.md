# CHANGELOG

All notable changes to **7FALL™** are documented here.

------------------------------------------------------------------------

## Version 2.1 --- July 2026

### New Features

-   Added **Fate Titles** based on the player's final score.
-   Added **Share My Fate** feature for sharing results.
-   Added **Play Again** button to the end-of-game screen.
-   Improved end-of-game presentation and messaging.

### Improvements

-   Updated game branding to **7FALL™**.
-   Updated canister artwork.
-   Updated electronic game timing to **1--4 Players • 3--5 Minutes per
    Player**.
-   Improved mobile sharing fallback behavior.
-   General UI polish and stability improvements.

------------------------------------------------------------------------

## Version 2.0 --- July 2026

### Major Changes

-   Rebranded from **SevenFall** to **7FALL™**.
-   Replaced application icons (Android, Apple, favicon).

-   Updated service-worker.js.
-   Refreshed project documentation.
-   Updated GitHub repository branding.
-   Updated website artwork and canister imagery.
